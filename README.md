# MP3 Downloader
Youtube, Spotify ve SoundCloud üzerinden dönüştürücü kullanarak mp3 müzik indirmenizi sağlar.

# Bilmeniz gerekenler
Uygulamayı uzun süredir kullanmıyorum belki dönüştürücü siteler artık çalışmıyor olabilir. Düzgün çalışması için kodları biraz düzenlemeniz gerekecektir. Onun dışında bir kaç ufak sorunları var, bazen indirilecek URL hatalı olursa bile indirme işlemi gerçekleşiyor. Bunu küçük bir kontrol ile düzeltebilirsiniz.
